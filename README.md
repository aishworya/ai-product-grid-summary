# AI-Based CRM Approval Assistant (POC)

## 📌 Overview

In CRM systems, pricing often depends on the **level of complexity** of a solution. However, complexity is subjective and inconsistently defined (e.g., "high design effort", "significant consulting").

This leads to:
- Incorrect pricing decisions
- Delays in approvals
- Manual effort for approvers reviewing large product grids

This project is a **Proof of Concept (POC)** that uses AI to:
- Summarize the solution from product grid data
- Validate whether the selected complexity level is appropriate
- Generate a clear approval email for faster decision-making

---

## 🚀 Features

- Product grid summarization
- AI-based complexity validation
- Confidence scoring
- Automated approval email generation

---

## 🧠 How It Works
User selects complexity in CRM
↓
Product Grid Data (CSV)
↓
Python Script (main.py)
↓
AI Processing (OpenAI API)

Summarization
Complexity validation
Confidence score
↓
Approval Email Output
